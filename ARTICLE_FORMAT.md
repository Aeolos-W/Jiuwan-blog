## 完整示例

以下是一篇完整文章的示例格式，你可以参考：

```
## 引言

今天我想分享一个有趣的问题。

> 数学是科学的皇后，而数论是数学的皇后。——高斯

## 主要内容

考虑以下方程：

$$
x^2 + y^2 = z^2
$$

这是一个**毕达哥拉斯方程**。它的解有以下特点：

1. 存在无穷多组整数解
2. 可以用参数化方法生成所有本原解
3. 与椭圆曲线有深刻联系

### 参数化解

对于任意整数 $m > n > 0$，令：

$$
x = m^2 - n^2, \quad y = 2mn, \quad z = m^2 + n^2
$$

则 $(x, y, z)$ 就是方程的一组解。

### 代码实现

\`\`\`python
def pythagorean_triples(limit):
    triples = []
    for m in range(2, limit):
        for n in range(1, m):
            x = m*m - n*n
            y = 2*m*n
            z = m*m + n*n
            triples.append((x, y, z))
    return triples
\`\`\`
```

## 发给我的格式

你只需要发给我以下内容即可：

1. **标题**：文章标题
2. **slug**：URL 标识（如 `my-new-post`，英文小写，不含空格）
3. **摘要**：一段简短描述
4. **分类**：选已有分类（Uncategorized / Career advice / On writing / Books / Applets）
5. **标签**：如 `{数学, 数论}`
6. **正文**：按上面的格式写的文章内容
